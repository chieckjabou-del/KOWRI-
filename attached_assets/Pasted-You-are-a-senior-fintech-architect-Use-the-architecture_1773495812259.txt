Create a complete production-ready fintech backend called KOWRI.

SYSTEM OVERVIEW

KOWRI is a digital financial infrastructure designed for African markets.  
It must support wallets, group savings (tontines), micro-credit, merchant payments, and financial reputation scoring.

The system must be scalable, secure and auditable.

TECH STACK

Node.js
Express.js
PostgreSQL
Docker
Docker Compose

ARCHITECTURE STYLE

Microservices architecture.

Each microservice must include:

- routes
- controllers
- services
- repositories
- database schema
- Dockerfile

COMMUNICATION

Services communicate using:

1. REST APIs for synchronous requests
2. Event-driven architecture for asynchronous communication

Use a message broker (Kafka or RabbitMQ).

FINANCIAL CORE RULES

The financial system must implement:

Double-entry accounting  
Atomic transactions  
Idempotent APIs  

The ledger must guarantee that:

Total debits = total credits.

LEDGER ARCHITECTURE

The core-ledger must implement event sourcing.

All financial operations must be stored as immutable events.

Examples:

deposit  
transfer  
loan disbursement  
loan repayment  
subscription charge  

The ledger must support full financial replay for auditing.
